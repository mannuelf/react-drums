# Welcome

React drums is a drum machine built with [React](https://reactjs.org/).

[Click here](https://react-drum-maschine.netlify.app/) to view live demo.

[Click here](https://github.com/mannuelf/react-drums) to download source code.

## Project Structure

📁 client/

📁 documentation/

📁 server/

## Start

### Local development

Three projects are managed as mono-repo with Yarn workspaces.

### Yarn install

```bash
yarn install
```

### Start client

```bash
yarn dev
```

### Test client

```bash
yarn test
```

## Build for production

```bash
yarn build
```
