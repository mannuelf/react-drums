## Getting started

Download repository: [https://github.com/mannuelf/react-drums](https://github.com/mannuelf/react-drums)

### Local development

#### Install Node.js dependencies

```bash
yarn install
```

#### Start local server

This will simultaniously start the React fronted, Apollo GraphQL server, Prisma Studio and documentation site respectively.

```bash
yarn dev
```
